# Gerenciamento-de-Inventario-de-Usarios-Dicionario-
Durante as aulas de Análise e Desenvolvimento de Sistemas (ADS), foi passada uma lista de exercícios práticos para desenvolver habilidades em Python.

Como parte da lista de exercícios em Python, este projeto (I2) teve como foco o uso prático de dicionários para representar e manipular informações em um sistema de controle de usuários.

O código permite:

Criar um dicionário com dados iniciais de um usuário

Adicionar novas chaves e valores

Verificar a existência de elementos com o operador in

Atualizar informações do dicionário

Objetivo: Simular a gestão de perfis de usuários em um ambiente de TI.

Habilidade desenvolvida: Manipulação de estruturas de dados (dicionários) e aplicação de operadores de associação em Python.
